# PersistentPad
A simple notepad that auto-saves locally on your device.

## Motivation

I built this simple notepad for my weekly anthropology assignments. For this course, we submit assignments through a university platform called Canvas. The platform uses a web interface with a text box (with formatting and word count) for submissions.

When brainstorming each assignment, I often wanted to take notes, but not directly on the website because I didn’t want to wait for login and page loading times. Also, without internet access, I couldn’t use it at all.

Because my university provides a Microsoft subscription, I initially used Microsoft Word. However, after one assignment, I wrote my notes in Word, closed my computer, and later found the document had been lost. I could have avoided this by enabling OneDrive, but I prefer not to rely on cloud syncing for these assignments.

I like the idea of auto-save, but I don’t want everything automatically uploaded to the cloud. I just want to open and close a file without worrying about manually saving every time.

## Design Philosophy

In general, I think some UI systems are too cloud-focused. While popular document tools avoid asking users to choose a save location upfront for simplicity, I believe having explicit local saving is sometimes a necessary trade-off for clarity and control.

This project is not meant to reinvent existing tools, but to explore the idea that simpler, local-first tools can still be valuable.

Future improvements could include formatting options and a word counter for essay-style writing.
